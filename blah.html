Common Issues:

  • $1M not showing: Check if P&L table has initial data or if starting cash is being loaded correctly
  • Can't buy/sell: Verify Supabase credentials, database connection, and table structure
  • No price data: Ensure daily_prices table has data for the symbols you're trying to trade
  • CORS errors: Check if running locally vs deployed - API endpoints might differ

Debug Steps:

  1. Verify environment variables (SUPABASE_URL, SUPABASE_KEY)
  2. Check database tables exist: holdings, daily_prices, pnl
  3. Verify daily_prices has recent data for test symbols
  4. Check browser console for JavaScript errors
  5. Test API endpoints individually
